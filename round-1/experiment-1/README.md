## Full Summary

Experiment: Validation of Pragmatic Tier–Fact Correctness Correlation (Assumption 1).

Objective: Test whether LLM-assigned pragmatic tiers (assertion=T1, presupposition=T2, implicature=T3) correlate with empirical fact correctness at |ρ| > 0.70 Spearman rank correlation.

Methodology:
- Corpus: 15 documents (5 legal, 5 news, 5 narrative), ~3000 chars each, curated inline.
- Fact extraction: Claude Haiku 4.5 via OpenRouter; 8 facts/document (3 assertions, 3 presuppositions, 2 implicatures) = 120 total facts.
- Correctness annotation: LLM-based strict verifier that distinguishes verbatim support vs. contradiction vs. inference.
- Tier classification: Extractor-assigned intended_tier (assertion/presupposition/implicature).
- Baseline: Raw LLM confidence score (0–100) correlation with correctness.

Primary Results:
- Spearman ρ (tier↔correctness) = -0.6796, |ρ| = 0.680, p < 0.0001 → verdict: INCONCLUSIVE (0.50 < |ρ| < 0.70).
- Tier accuracy gradient confirmed: T1 Assertion=100%, T2 Presupposition=49%, T3 Implicature=17%.
- Baseline (confidence↔correctness): ρ = 0.722; tier advantage = -0.043 (confidence is marginally better predictor).
- Per-document type: legal ρ=-0.718, news ρ=-0.607, narrative ρ=-0.706.

Key Finding: Pragmatic tier is a moderate predictor of fact correctness but does not reach the r>0.70 CONFIRM threshold under strict verification. The expected accuracy ordering (T1>T2>T3) is strongly confirmed. Raw LLM confidence scores are marginally better predictors, suggesting pragmatic tier classification adds linguistic interpretability but not additional predictive power over confidence alone.

Baseline comparison: Two baselines implemented — (1) raw LLM confidence, (2) expected tier accuracy from document type. Both are compared against the tier-based method.

Outputs: 120 annotated facts with tier/correctness labels, scatter plots, misclassification analysis, calibration recommendations per document type, JSONL supplementary data.
