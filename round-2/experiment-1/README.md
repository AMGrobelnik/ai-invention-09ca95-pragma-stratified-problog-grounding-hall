## Layman Summary

Tests whether assigning different confidence scores to direct vs implied linguistic statements improves AI reasoning accuracy on logic puzzles, using formal statistical tests to compare against standard AI baselines.

## Full Summary

This experiment implements and rigorously evaluates the Pragma-Stratified ProbLog hypothesis: that classifying linguistic facts by pragmatic tier (assertion p=0.93, presupposition p=0.78, implicature p=0.61) improves neuro-symbolic reasoning accuracy on two established benchmarks.

**Datasets used:** 200 CLUTRR (kinship reasoning, n=12,064 total) + 200 RuleTaker (rule-based entailment, n=49,999 total), stratified sampling.

**Five methods implemented and compared:**
1. RAG baseline: BM25 retrieval, top-1 sentence prediction
2. Chain-of-Thought (CoT): LLM (llama-3.1-8b-instruct via OpenRouter) with step-by-step prompting
3. Flat FOL ablation: rule-based kinship/forward-chain with uniform p=0.5 (no tier weighting)
4. Pragma-Stratified ProbLog (proposed): same symbolic engine but tier-weighted probabilities
5. LLM-TRes: LLM + symbolic verification hybrid

**Stage 1 — Tier-correctness Spearman correlation:** ρ=-0.051, 95% CI=[-0.135, 0.034], p=0.241, n=537 fact-correctness pairs. Tier ordinal encoding (assertion=3, presupposition=2, implicature=1) vs binary correctness. Result: INCONCLUSIVE (p≥0.10). Per-domain stratification by RuleTaker config (NatLang, depth-1 through depth-5). Comparison to confidence baseline (ρ=0.7226): tier ρ is weaker.

**Stage 2 — McNemar benchmark comparison (n=400 combined):**
- RAG: 32.2% accuracy, 95% CI=[0.279, 0.368]
- CoT: 34.3% accuracy, 95% CI=[0.298, 0.390]
- Flat FOL: 39.5% accuracy, 95% CI=[0.348, 0.444]
- Pragma-Stratified ProbLog: 39.5% accuracy, 95% CI=[0.348, 0.444]
- LLM-TRes: 40.5% accuracy, 95% CI=[0.358, 0.454]
- Primary McNemar test (pragma_problog vs CoT): χ²=3.67, p=0.0554, MARGINALLY SIGNIFICANT (p<0.10), Cohen's h=0.11 (small effect), discordant pairs=109
- 10 pairwise McNemar tests with contingency tables, Wilson CIs, and Cohen's h for all method pairs

**Stage 3 — Hallucination bound power analysis (n=295 proof pairs):**
- Pearson r(HB, correctness)=0.239, 95% CI=[0.128, 0.344], p=3.4e-5
- SIGNIFICANT (p<0.05), post-hoc power=98.6% (ADEQUATE)
- HB validated as proxy for hallucination risk with sufficient statistical power
- n_required for 80% power at r=0.239: n=136 (current n=295, well above threshold)

**Overall hypothesis status: REFINE** — marginal benchmark advantage, strong HB validation, inconclusive tier-correctness correlation.
